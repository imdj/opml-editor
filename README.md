# OPML Online editor

A simple online OPML editor. Create, edit, merge, save OPML files easily. Visit [opml.imadij.com](https://opml.imadij.com).

Written in SvelteKit.

## Features
- Add new feed
- Merge multiple OPML files
- Remove duplicates
